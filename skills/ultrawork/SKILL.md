# Ultrawork Skill

## 기능

- 독립적인 작업을 여러 에이전트가 동시에 처리
- 전문 에이전트에게 즉시 작업 라우팅
- 긴 작업은 백그라운드 실행
- 모든 작업 완료 및 검증까지 지속

```
[ ] 작업 분석 및 분해
```
