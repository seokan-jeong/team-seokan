# Ralph Skill

## 기능

- 작업 완료까지 무한 반복 시도
- 에러 발생 시 자동 복구 시도
- TODO 리스트로 진행 상황 추적
- 액션가면(Reviewer)의 최종 검증

## Ralph 루프

1. TODO 리스트 확인
2. 다음 작업 수행
3. 결과 검증
4. 실패 시 → 원인 분석 → 재시도
5. 성공 시 → 다음 작업으로
6. 모든 작업 완료 → 최종 검증
7. 검증 실패 → 수정 후 재검증

## 워크플로우 체크리스트

```
[ ] 작업 목록 초기화
[ ] 현재 작업 수행
[ ] 작업 결과 검증
[ ] 모든 작업 완료
[ ] 액션가면 최종 검증
```

## 완료 조건

다음 모든 조건 충족 시에만 완료:
- TODO 리스트의 모든 항목 완료
- 빌드/테스트 통과
- 액션가면의 리뷰 승인

**조건 미충족 시 자동으로 계속 작업**
