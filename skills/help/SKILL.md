## 메모리 시스템

Team-Seokan은 사용할수록 똑똑해집니다:
- 자동으로 패턴, 선호도, 실수를 학습
- 학습된 내용을 다음 작업에 활용
- `/memories`로 학습 내용 확인
- `/learn`으로 수동 학습 추가
