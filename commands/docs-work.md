# Docs-Work Command

Execute the docs-work skill for issue-based workflow management.

See `skills/docs-work/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:docs-work ISSUE-xxx
```

## Quick Start

1. Creates/reviews `shinchan-docs/ISSUE-xxx/REQUESTS.md`
2. Phase-based work with review cycles
3. Auto-generates RETROSPECTIVE.md + IMPLEMENTATION.md on completion
