# Setup Command

Runs first-time installation onboarding and verifies plugin health.

## Usage

```
/team-shinchan:setup
```

## What It Does

1. Displays plugin version
2. Checks hooks registration status
3. Initializes `.shinchan-docs/` directory
4. Shows agent roster with memory status
5. Runs full validation suite
6. Outputs summary report
